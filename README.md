# WhatsApp Chat Viewer 📦

A lightweight, privacy-first web application to view and analyze your exported WhatsApp chat backups entirely offline. 

Nothing leaves your machine; all parsing and rendering happen strictly inside your web browser.

## ✨ Features
* **100% Offline & Private:** Your data never touches a server.
* **Smart Parsing:** Supports both Android and iOS `.txt` chat exports, as well as bulk `.zip` backup folders.
* **Modern UI:** Clean, responsive WhatsApp-inspired theme with a built-in dark/light design.
* **Chat Analytics:** Instantly see total messages, media attachments, active days, and participant counts.
* **Full Text Search:** Search within specific conversations with highlighted keyword results.

## 🚀 How to Use It

### Option 1: Use it Live (Recommended)
You can open and use the application directly in your browser here: 
👉 **https://ahmadanka1.github.io/whatsapp-chat-viewer/**

### Option 2: Run Locally
1. Go to the **Releases** section on the right side of this page.
2. Download the latest source code package.
3. Extract the contents and double-click the `index.html` file to open it in any modern browser.
4. Drag and drop your exported WhatsApp `.txt` or `.zip` archive to start viewing!
